Chrome Extension for LinkedIn Lead Tracking

A custom tool I developed to streamline the lead generation process for sales development representatives, allowing for efficient data collection and management directly from LinkedIn profiles.

JavaScript HTML/CSS Chrome API LinkedIn API

Project Overview

As a Sales Development Representative, I identified a significant bottleneck in the lead generation process: manually collecting and transferring prospect information from LinkedIn to our CRM system. To solve this problem, I developed a Chrome extension that:

  • Automatically extracts key information from LinkedIn profiles
  • Stores lead data in cloud for easy access
  • Includes note-taking capabilities for each prospect

The extension has significantly improved my prospecting efficiency, reducing the time spent on data entry by approximately 40%.

Key Features

  • One-Click Data Extraction: Extract contact information, current role, company details with a single click
  • Custom Tags and Categories: Organize leads with custom tags for better follow-up management
  • Export Functionality: Export data in Google Sheets format for easy import to CRM system
  • Activity Tracking: Log communication history and follow-up reminders

Development Process

I developed this extension over a while working full-time as an SDR. The process included:

  1. Research & Planning: Investigating LinkedIn's DOM structure and identifying key information fields
  2. Extension Architecture: Designing the data flow and storage approach
  3. UI Design: Creating a clean, intuitive interface that doesn't interrupt the normal browsing experience
  4. Development: Building the core functionality using JavaScript and Chrome's extension APIs
  5. Testing: Extensive testing with actual prospects to ensure accuracy and reliability

Results & Impact

Since implementing this tool in my workflow:

  • Reduced time spent on data entry by 40%
  • Increased daily prospecting capacity by 30%
  • Improved data accuracy by eliminating manual entry errors
  • Enhanced follow-up consistency with integrated reminders
